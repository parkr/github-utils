# github-team-radar

What's everyone on your team up to?

```text
Usage of github-team-radar:
  -create
    	Post the issue to GitHub
  -mention string
    	The user or team to mention at the top of the radar issue
  -owner string
    	The repository owner the radar issue should be written to
  -repo string
    	The repository owner the radar issue should be written to
```
